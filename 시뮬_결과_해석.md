# 자가대전 시뮬 결과 해석 (5000판)

## 신뢰할 수 있는 결론 (1000판+)
- 치나츠 60.8% / 네루 60.6% / 미네 63.6% — 강세 확정
- 카요코 42.4% (11717판) — AI 과대평가, 실전 약함. 밸류 재검토 필요
- 키쿄 40.5% (2779판) — 연사+보호막인데 승률 낮음. 스탯 약함
- 카야 33.0% (4147판) — 5티어인데 하위권. 개전 능력 실효 확인 필요
- 아리스 38.4% (740판) — 관통 9/2지만 체력 2라 빨리 죽음

## 샘플 부족 (30판 이하, 판단 보류)
- 아즈사 79.3% / 아코 69% / 카즈사 67.6% / 히후미 65.2%

1. AI 구매 편향 — 선호 카드 판수 많고 비선호 카드 판수 적음
2. 카드 시너지 미반영
3. 메타 반응형 전략 없음

1. 카요코 밸류 하향 여부
2. 카야 개전 능력 코드 확인 (버그인지 설계인지)
3. 아리스 스탯 조정 여부
4. 샘플 적은 카드 추가 판 돌리기
5. 학습 결과 실게임 AI 반영 방식
